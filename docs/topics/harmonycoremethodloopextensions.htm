Method Loop Extensions

 

Harmony Core Method Loop Extensions

 

Expansion Tokens

 

<HARMONYCORE_BRIDGE_RETURN_TYPE>

 

Inserts the Synergy .NET data type to be used when processing a method return type following a Harmony Core Traditional Bridge method call. The logic used when determining the Synergy .NET type to use follows the guidelines defined in the Synergy Method Catalog for the parameter, including any coercion rules that may be defined.

 

Alternate forms: none

 

<HARMONYCORE_BRIDGE_RETURN_TYPE_CS>

 

Inserts the C# data type to be used when processing a method return type following a Harmony Core Traditional Bridge method call. The logic used when determining the C# type to use follows the guidelines defined in the Synergy Method Catalog for the parameter, including any coercion rules that may be defined.

 

Alternate forms: none

 

 

<HARMONYCORE_SHORT_METHOD_NAME>

 

Inserts a possibly mangled name for methods with long names.

 

Alternate forms: none
