Method Loop Extensions

 

Harmony Core Method Loop Extensions

 

Expansion Tokens

 

<HARMONYCORE_BRIDGE_RETURN_TYPE>

 

Inserts the .NET data type to be used when processing a method return type following a Harmony Core Traditional Bridge method call. The logic used when determining the .NET type to use follows the guidelines defined in the Synergy Method Catalog for the parameter, including any coercion rules that may be defined.

 

Alternate forms: none
