Generic Extensions

 

Harmony Core Generic Extensions

 

Expansion Tokens

 

None

 

Expressions

 

<IF HARMONYCORE_RELATIONS_ENABLED>

 

Does the structure have relations enabled?

 

Relations can be enabled for all structures via the CodeGen define ENABLE_RELATIONS, or for a specific structure by adding the text HARMONYCORE_RELATIONS_ENABLED=true; or HARMONYCORE_RELATIONS_ENABLED=false; to a structures long description or user-defined text field.

 

<IF HARMONYCORE_RELATIONS_VALIDATION_ENABLED>

 

Does the structure have relation validation enabled?

 

Relation validation can be enabled for all structures via the CodeGen define ENABLE_RELATIONS_VALIDATION, or for a specific structure by adding the text HARMONYCORE_RELATIONS_VALIDATION_ENABLED=true; or HARMONYCORE_RELATIONS_VALIDATION_ENABLED=false; to a structures long description or user-defined text field.
