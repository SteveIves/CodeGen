FIELD_TYPE_NAME

 

Inserts a string indicating the Synergy data type of field.

Usage

<FIELD_TYPE_NAME>

Possible values

•ALPHA

•AUTO SEQUENCE

•AUTO TIME

•BINARY

•DATE

•DECIMAL

•IMPLIED

•INTEGER

•JULIAN

•TIME

•USER ALPHA

•USER DATE

•USER NUMERIC

Alternate forms

None
