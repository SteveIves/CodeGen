Template Browser Utility

 

The Template Browser utility allows you to browse (and in some cases edit) CodeGen template files.

You can start the utility from a Start Menu shortcut, and you will see three tabs containing different types of templates.

•CodeGen Sample Templates (if installed)

•Symphony Framework Sample Templates (if installed)

•Your own templates.

A menu option allows you to specify the location of your own template files.
