METHOD_RETURN_DATE_FORMAT

 

Inserts the date format associated with the method return value.

 

Usage
 

<METHOD_RETURN_DATE_FORMAT>

Possible values

•None

•YYYYMMDD

•YYMMDD

•YYJJJ

•YYYYJJJ

•HHMM

•HHMMSS

•YYYYMMDDHHMISS
 

Alternate forms

None
