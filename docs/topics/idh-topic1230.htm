FIELD_VBTYPE

 

Inserts a string indicating the VB.NET data type of the field.

Usage

<FIELD_VBTYPE>

Possible values

•Boolean

•Byte

•DateTime

•Decimal

•Integer

•Long

•Nullable(Of DateTime)

•Short

•String

•System.Double

•System.SByte

•System.Single

•System.UInt16

•System.UInt32

•System.UInt64

Alternate forms

None
