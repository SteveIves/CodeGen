DateFormat

 

 

The DateFormat enumeration defines the various format for date parameters for methods defined in a method catalog interface.

 

public enum DateFormat

    None

    YYYYMMDD

    YYMMDD

    YYJJJ

    YYYYJJJ

    HHMM

    HHMMSS

    YYYYMMDDHHMISS

endenum
