Harmony Core CodeGen Extensions

 

Harmony Core CodeGen Extensions are a collection of custom CodeGen extensions that have been developed by the Harmony Core development team specifically for use when generating code that utilizes Harmony Core.

 

These extensions are only available when the Harmony Core Extensions feature is selected during CodeGen installation. By default these extensions are installed.
