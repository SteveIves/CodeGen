Processing UI Toolkit Windows

 

CodeGen can process some metadata that originates from UI Toolkit Input Window definitions located in  UI Toolkit window script files. This functionality is enabled via the -w command line option.

 

This feature allows you to generate basic UI in the form of forms for windows, web, or mobile applications.

 

Window Tokens are tokens that can only be used when processing a UI Toolkit input window.
