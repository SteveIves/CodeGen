Interface Loop Expression Tokens

 

Interface loop expressions are expressions that can only be used within the context of an interface loop. CodeGen supports interface loop expressions via the <IF expression> and </IF [expression]> tags. Note that loop utility expressions can also be used within interface loops.

 

Expression

Description

<IF CUSTOM_INTERFACE_expression>

Custom interface loop expressions allow you to test custom conditions based on custom interface tokens defined in a method catalog extensions file. Note that custom expressions are not case sensitive, always being converted to uppercase for comparison.
