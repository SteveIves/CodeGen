KEY_NULLTYPE

 

Inserts the keys null key type.  The token should only be used within a <IF NULLKEY> expression.

Usage

<KEY_NULLTYPE>

Possible values

•NONE

•REPLICATE

•NOREPLICATE

•SHORT

Alternate forms

•<key_nulltype>
