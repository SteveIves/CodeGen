CodeGen Tweaks

 

Tweak Name

 

Purpose

GROUP_PREFIX_ALWAYS

Causes any group prefixes to be automatically applied to group fields, even if the "Use by compiler" option is not set.

PARAMDEFSTR

 

Causes the <PARAMETER_ATTRIBUTE>, <PARAMETER_DEFINITION>, <PARAMETER_DEFINITION_NOARRAY>, <PARAMETER_STRUCTURE>, <PARAMETER_STRUCTURE_NOPLURAL> and <PARAMETER_STRUCTURE_PLURAL> tokens to prefix structure names with "STR_".

 

SQLNAMEN$$ID

 

Causes $$ in field names to be replaced by the text "Id" when using the <FIELD_SQLNAME> field loop token.

SQLNAMENO$

 

Causes any $ characters in field names to be suppressed when using the <FIELD_SQLNAME> field loop token.
