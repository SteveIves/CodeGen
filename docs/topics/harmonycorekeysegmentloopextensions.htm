Key Segment Loop Extensions

 

Harmony Core Key Segment Loop Extensions

 

Expansion Tokens

 

<HARMONYCORE_SEGMENT_DATATYPE>

 

If the field associated with the current key segment has a custom data type (as signified by the presence of a special token in the format of HARMONYCORE_CUSTOM_FIELD_DATATYPE=type;) then that custom data type is inserted. Otherwise the fields actual Synergy .NET data type (<SEGMENT_SNTYPE>) is inserted.

 

Alternate forms: none

 

Expressions

 

<IF HARMONYCORE_CUSTOM_SEGMENT_DATATYPE>

 

Tests whether the field associated with the current key segment has a custom data type (as signified by the presence of a special token in the format of HARMONYCORE_CUSTOM_FIELD_DATATYPE=type;)
