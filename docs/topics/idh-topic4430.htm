Relation Key Segment Loop Expansion Tokens

 

All key segment loop expansion tokens and field loop expansion tokens (for field segments) are available in Relation Key Segment Loops, and additionally the following tokens are available:

 

Token

Description

<LITERAL_SEGMENT_CSTYPE>

 

The C# data type suitable to represent the literal segment. This data type is derived by looking at the matching segment on the "to key" side of the relationship.

 

<LITERAL_SEGMENT_SNTYPE>

 

The Synergy .NET data type suitable to represent the literal segment. This data type is derived by looking at the matching segment on the "to key" side of the relationship.

<LITERAL_SEGMENT_VALUE>

 

The literal value of the key segment. This will be either a quoted string or a numeric literal value.
